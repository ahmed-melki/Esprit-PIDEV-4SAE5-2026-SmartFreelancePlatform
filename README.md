# 🎫 Event Management System - Application de Gestion d'Événements

## 🚀 Description

Application complète de gestion d'événements développée avec **Spring Boot** (backend) et **Angular** (frontend). Permet aux administrateurs de créer, modifier, supprimer des événements, et aux utilisateurs de consulter et s'inscrire aux événements.

## ✨ Fonctionnalités

### Backend (Spring Boot)
- ✅ CRUD complet des événements
- ✅ Gestion des catégories et lieux
- ✅ Système d'authentification JWT
- ✅ Gestion des inscriptions aux événements
- ✅ Génération de rapports PDF
- ✅ Tests unitaires (JUnit 5 + Mockito)
- ✅ API REST documentée (Swagger/OpenAPI)

### Frontend (Angular)
- ✅ Liste des événements avec filtres
- ✅ Création et modification d'événements
- ✅ Inscription aux événements
- ✅ Dashboard administrateur
- ✅ Interface responsive
- ✅ Tests unitaires (Jasmine + Karma)
