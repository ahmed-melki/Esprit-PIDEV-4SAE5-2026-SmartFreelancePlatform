# SmartFreelance – Freelance Platform Application

## Overview
This project was developed as part of the PIDEV – 4th Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).

It consists of a full-stack web application that connects freelancers with clients, enabling project posting, proposal submissions, contract management, and real-time communication.

## Tech Stack
- Angular
- Spring Boot
- MySQL
- Microservices
- REST API
