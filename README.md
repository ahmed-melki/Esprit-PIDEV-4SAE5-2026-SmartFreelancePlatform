# 📝 Blog Service - Application de Gestion de Blog

## 🚀 Description

Application de gestion de blog développée avec **Spring Boot** (backend) et **Angular** (frontend). Permet aux utilisateurs de créer, modifier, supprimer des articles, gérer des réactions (likes/dislikes), et signaler des contenus inappropriés.

## ✨ Fonctionnalités

### Backend (Spring Boot)
- ✅ CRUD complet des articles
- ✅ Système de réactions (Like/Dislike)
- ✅ Badges automatiques selon le nombre de likes
- ✅ Signalement d'articles
- ✅ Tests unitaires (JUnit 5 + Mockito)

### Frontend (Angular)
- ✅ Liste des articles avec recherche
- ✅ Création et modification d'articles
- ✅ Système de likes/dislikes avec animations
- ✅ Signalement d'articles
- ✅ Interface responsive
- ✅ Tests unitaires (Jasmine + Karma)
